Key Import Guidelines

When importing keys, please keep in mind:

  • Symmetric keys (AES, 3DES) should be base64-encoded strings.
  • RSA keys should be in PEM format (base64-encoded with header/footer).
  • For Diffie-Hellman keys, include both the public and private components.
  • Key sizes must be standard for the algorithm (e.g., AES-128: 16 bytes, AES-256: 32 bytes).
  • Imported keys will be stored in your account's key repository and can be used like any other key.

Important: Make sure you trust the source of any keys you import!
