# Cryptography Web Application

A secure Flask-based web application that allows users to upload files and perform cryptographic operations through a simple web interface.

## Features
- User authentication and account management
- File upload and storage
- File encryption and decryption
- Support for AES, 3DES, and RSA
- Secure hashing for integrity verification
- Key generation and key management options

## Technologies Used
- Flask
- Python
- HTML/CSS
- SQLite
- Cryptography libraries

## Purpose
This project was developed for an Information Security course to demonstrate practical implementation of modern cryptographic services in a web application environment.

## Notes
The application supports both symmetric and asymmetric encryption workflows and allows users to securely manage keys for different cryptographic operations.
